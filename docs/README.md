## 학습 가이드

### 면접 준비 우선순위

```
★★★ 필수    운영체제, 네트워크, 자료구조와 알고리즘, 데이터베이스
★★☆ 중요    소프트웨어 아키텍처, 컴퓨터 구조, 분산 시스템
★☆☆ 직무별  웹 개발, 클라우드, 보안, AI/ML, 인프라 도구
```

### 추천 학습 순서

```
[CS 기초] → [컴퓨터 구조] → [운영체제] → [자료구조/알고리즘]
                                              ↓
[분산 시스템] ← [소프트웨어 공학] ← [데이터베이스] ← [네트워크]
      ↓
[인프라 도구] → [클라우드] → [직무별 심화]
```
